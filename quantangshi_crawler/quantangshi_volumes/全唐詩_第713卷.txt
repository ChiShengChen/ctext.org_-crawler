全唐詩 第713卷
==================================================

1. 陳情獻中丞
   作者: 喻坦之著
   內容:
陳情獻中... :
孤拙竟何營，徒希折桂名。始終誰肯薦，得失自難明。
陳情獻中... :
貢乏雄文獻，歸無瘠土耕。滄江長髮夢，紫陌久慚行。
陳情獻中... :
意縱求知切，才惟懼鑒精。五言非琢玉，十載看遷鶯。
陳情獻中... :
取進心甘鈍，傷嗟骨每驚。塵襟痕積淚，客鬢白新莖。
陳情獻中... :
顧盼身堪教，吹噓羽覺生。依門情轉切，荷德力須傾。
陳情獻中... :
獎善猶憐貢，垂恩必不輕。從茲便提挈，雲路自生榮。
------------------------------

2. 長安雪後
   作者: 喻坦之著
   內容:
長安雪後:
碧落雲收盡，天涯雪霽時。草開當井地，樹折帶巢枝。
長安雪後:
野渡滋寒麥，高泉漲禁池。遙分丹闕出，迥對上林宜。
長安雪後:
宿片攀簷取，凝花就砌窺。氣淩禽翅束，凍入馬蹄危。
長安雪後:
北想連沙漠，南思極海涯。冷光兼素彩，向暮朔風吹。
------------------------------

3. 送友人游東川
   作者: 喻坦之著
   內容:
送友人游... :
食盡須分散，將行幾願留。春兼三月閏，人擬半年遊。
送友人游... :
風俗同吳地，山川擁梓州。思君登棧道，猿嘯始應愁。
------------------------------

4. 題樟亭驛樓
   作者: 喻坦之著
   內容:
題樟亭驛... :
危檻倚山城，風帆檻外行。日生滄海赤，潮落浙江清。
題樟亭驛... :
秋晚遙峰出，沙幹細草平。西陵煙樹色，長見伍員情。
------------------------------

5. 大樑送友人東游
   作者: 喻坦之著
   內容:
大樑送友... :
自古東西路，舟車此地分。河聲梁苑夜，草色楚田曛。
大樑送友... :
雁已多南去，蟬猶在此聞。聖朝無諫獵，何計謁明君。
------------------------------

6. 送友人游蜀
   作者: 喻坦之著
   內容:
送友人游... :
為儒早得名，為客不憂程。春盡離丹闕，花繁到錦城。
送友人游... :
雪消巴水漲，日上劍關明。預想回來樹，秋蟬已數聲。
------------------------------

7. 留別友人書齋
   作者: 喻坦之著
   內容:
留別友人... :
相見不相睽，一留日已西。軒涼庭木大，巷僻鳥巢低。
留別友人... :
背俗修琴譜，思家話藥畦。卜鄰期太華，同上上方梯。
------------------------------

8. 題耿處士林亭
   作者: 喻坦之著
   內容:
題耿處士... :
身向閑中老，生涯本豁然。草堂山水下，漁艇鳥花邊。
題耿處士... :
窺井猿兼鹿，啼林鳥雜蟬。何時人事了，依此亦高眠。
------------------------------

9. 商于逢友人
   作者: 喻坦之著
   內容:
商于逢友... :
行役何時了，年年骨肉分。春風來漢棹，雪路入商雲。
商于逢友... :
水險溪難定，林寒鳥異群。相逢聊坐石，啼狖語中聞。
------------------------------

10. 灞上逢故人
   作者: 喻坦之著
   內容:
灞上逢故... :
花落杏園枝，驅車問路岐。人情誰可會，身事自堪疑。
灞上逢故... :
岳雨狂雷送，溪槎漲水吹。家山如此景，幾處不相隨。
------------------------------

11. 發浙江
   作者: 喻坦之著
   內容:
發浙江:
島嶼遍含煙，煙中濟大川。山城猶轉漏，沙浦已搖船。
發浙江:
海曙霞浮日，江遙水合天。此時空闊思，翻想涉窮邊。
------------------------------

12. 晚泊盱眙
   作者: 喻坦之著
   內容:
晚泊盱眙:
廣葦夾深流，蕭蕭到海秋。宿船橫月浦，驚鳥繞霜洲。
晚泊盱眙:
雲濕淮南樹，笳清泗水樓。徒懸鄉國思，羈跡尚東遊。
------------------------------

13. 歸江南
   作者: 喻坦之著
   內容:
歸江南:
歸日值江春，看花過楚津。草晴蟲網遍，沙曉浪痕新。
歸江南:
蓮葉初浮水，鷗雛已狎人。漁心慚未遂，空厭路岐塵。
------------------------------

14. 代北言懷
   作者: 喻坦之著
   內容:
代北言懷:
困馬榆關北，那堪落景催。路行沙不絕，風與雪兼來。
代北言懷:
草得春猶白，鴻侵夏始回。行人莫遠入，戍角有餘哀。
------------------------------

15. 春遊曲江
   作者: 喻坦之著
   內容:
春遊曲江:
誤入杏花塵，晴江一看春。菰蒲雖似越，骨肉且非秦。
春遊曲江:
曲岸藏翹鷺，垂楊拂躍鱗。徒憐汀草色，未是醉眠人。
------------------------------

18. 晚泊富春寄友人
   作者: 喻坦之著
   內容:
晚泊富春... :
江鐘寒夕微，江鳥望巢飛。木落山城出，潮生海棹歸。
晚泊富春... :
