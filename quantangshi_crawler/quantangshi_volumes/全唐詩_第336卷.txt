1. 元和聖德詩
   作者: 韓愈著
   內容:
元和聖德... :
皇帝即阼，物無違拒。曰暘而暘，曰雨而雨。維是元年，有盜在夏。欲覆其州，以踵近武。皇帝曰嘻，豈不在我。
元和聖德... :
負鄙為艱，縱則不可。出師征之，其眾十旅。軍其城下，告以福禍。腹敗枝披，不敢保聚。擲首陴外，降幡夜豎。
元和聖德... :
疆外之險，莫過蜀土。韋皋去鎮，劉辟守後。血人於牙，不肯吐口。開庫啖士，曰隨所取。汝張汝弓，汝鼓汝鼓。
元和聖德... :
汝為表書，求我帥汝。事始上聞，在列咸怒。皇帝曰然，嗟遠士女。苟附而安，則且付與。讀命於庭，出節少府。
元和聖德... :
朝發京師，夕至其部。辟喜謂黨，汝振而伍。蜀可全有，此不當受。萬牛臠炙，萬甕行酒。以錦纏股，以紅帕首。
元和聖德... :
有恇其凶，有餌其誘。其出穰穰，隊以萬數。遂劫東川，遂據城阻。皇帝曰嗟，其又可許。爰命崇文，分卒禁禦。
元和聖德... :
有安其驅，無暴我野。日行三十，徐壁其右。辟党聚謀，鹿頭是守。崇文奉詔，進退規矩。戰不貪殺，擒不濫數。
元和聖德... :
四方節度，整兵頓馬。上章請討，俟命起坐。皇帝曰嘻，無汝煩苦。荊並洎梁，在國門戶。出師三千，各選爾醜。
元和聖德... :
四軍齊作，殷其如阜。或拔其角，或脫其距。長驅洋洋，無有齟齬。八月壬午，辟棄城走。載妻與妾，包裹稚乳。
元和聖德... :
是日崇文，入處其宇。分散逐捕，搜原剔藪。辟窮見窘，無地自處。俯視大江，不見洲渚。遂自顛倒，若杵投臼。
元和聖德... :
取之江中，枷脰械手。婦女累累，啼哭拜叩。來獻闕下，以告廟社。周示城市，咸使觀睹。解脫攣索，夾以砧斧。
元和聖德... :
婉婉弱子，赤立傴僂。牽頭曳足，先斷腰膂。次及其徒，體骸撐拄。末乃取辟，駭汗如寫。揮刀紛紜，爭刌膾脯。
元和聖德... :
優賞將吏，扶珪綴組。帛堆其家，粟塞其庾。哀憐陣沒，廩給孤寡。贈官封墓，周匝宏溥。經戰伐地，寬免租簿。
元和聖德... :
施令酬功，急疾如火。天地中間，莫不順序。幽恒青魏，東盡海浦。南至徐蔡，區外雜虜。怛威赧德，踧踖蹈舞。
元和聖德... :
掉棄兵革，私習簋簠。來請來覲，十百其耦。皇帝曰籲，伯父叔舅。各安爾位，訓厥甿畝。正月元日，初見宗祖。
元和聖德... :
躬執百禮，登降拜俯。薦于新宮，視瞻梁梠。戚見容色，淚落入俎。侍祠之臣，助我惻楚。乃以上辛，於郊用牡。
元和聖德... :
除于國南，鱗筍毛簴。廬幕周施，開揭磊砢。獸盾騰拏，圓壇帖妥。天兵四羅，旗常婀娜。駕龍十二，魚魚雅雅。
元和聖德... :
宵升於丘，奠璧獻斝。眾樂驚作，轟豗融冶。紫焰噓呵，高靈下墮。群星從坐，錯落侈哆。日君月妃，煥赫婐vi。
元和聖德... :
瀆鬼蒙鴻，嶽祗嶪峨。飫沃膻薌，產祥降嘏。鳳凰應奏，舒翼自拊。赤麟黃龍，逶陀結糾。卿士庶人，黃童白叟。
元和聖德... :
踴躍歡呀，失喜噎歐。幹清坤夷，境落褰舉。帝車回來，日正當午。幸丹鳳門，大赦天下。滌濯剗磢，磨滅瑕垢。
元和聖德... :
續功臣嗣，拔賢任耇。孩養無告，仁滂施厚。皇帝神聖，通達今古。聽聰視明，一似堯禹。生知法式，動得理所。
元和聖德... :
天錫皇帝，為天下主。並包畜養，無異細鉅。億載萬年，敢有違者？皇帝儉勤，盥濯陶瓦。斥遣浮華，好此綈紵。
元和聖德... :
敕戒四方，侈則有咎。天錫皇帝，多麥與黍。無召水旱，耗於雀鼠。億載萬年，有富無窶。皇帝正直，別白善否。
元和聖德... :
擅命而狂，既翦既去。盡逐群奸，靡有遺侶。天錫皇帝，厖臣碩輔。博問遐觀，以置左右。億載萬年，無敢餘侮。
元和聖德... :
皇帝大孝，慈祥悌友。怡怡愉愉，奉太皇后。浹於族親，濡及九有。天錫皇帝，與天齊壽。登茲太平，無怠永久。
元和聖德... :
億載萬年，為父為母。博士臣愈，職是訓詁。作為歌詩，以配吉甫。

2. 琴操十首：將歸操
   作者: 韓愈著
   內容:
琴操十首... :
（孔子之趙，聞殺鳴犢作。趙殺鳴犢，孔子臨河，歎而作歌曰：秋之水兮風揚波，舟楫顛倒更相加，歸來歸來胡為斯）
琴操十首... :
秋之水兮，其色幽幽；我將濟兮，不得其由。涉其淺兮，石齧我足；乘其深兮，龍入我舟。我濟而悔兮，將安歸尤。歸兮歸兮，無與石鬥兮，無應龍求。

3. 琴操十首：猗蘭操
   作者: 韓愈著
   內容:
琴操十首... :
（孔子傷不逢時作。古琴操雲：習習谷風，以陰以雨。
琴操十首... :
之子於歸，遠送於野。何彼蒼天，不得其所。逍遙九州，無有定處。世人暗蔽，不知賢者。年紀逝邁，一身將老）
琴操十首... :
蘭之猗猗，揚揚其香。不采而佩，于蘭何傷。
琴操十首... :
今天之旋，其曷為然。我行四方，以日以年。
琴操十首... :
雪霜貿貿，薺麥之茂。子如不傷，我不爾覯。
琴操十首... :
薺麥之茂，薺麥之有。君子之傷，君子之守。

   作者: 韓愈著
   內容:

（牧犢子七十無妻，見雉雙飛，感之而作。本詞雲：
琴操十首... :
雉朝飛兮鳴相和，雌雄群遊兮山之阿。
琴操十首... :
我獨何命兮未有家，時將暮兮可奈何，嗟嗟暮兮可奈何）
琴操十首... :
雉之飛，於朝日。群雌孤雄，意氣橫出。當東而西，當啄而飛。隨飛隨啄，群雌粥粥。嗟我雖人，曾不如彼雉雞。生身七十年，無一妾與妃。

13. 謝自然詩
   作者: 韓愈著
   內容:
