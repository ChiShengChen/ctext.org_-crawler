1. 越中秋懷
   作者: 李白著
   內容:
越中秋懷:
越水繞碧山，周回數千里。乃是天鏡中，分明畫相似。
越中秋懷:
愛此從冥搜，永懷臨湍遊。一為滄波客，十見紅蕖秋。
越中秋懷:
觀濤壯天險，望海令人愁。路遐迫西照，歲晚悲東流。
越中秋懷:
何必探禹穴，逝將歸蓬丘。不然五湖上，亦可乘扁舟。

2. 效古二首
   作者: 李白著
   內容:
效古二首:
朝入天苑中，謁帝蓬萊宮。青山映輦道，碧樹搖蒼空。
效古二首:
謬題金閨籍，得與銀台通。待詔奉明主，抽毫頌清風。
效古二首:
歸時落日晚，躞蹀浮雲驄。人馬本無意，飛馳自豪雄。
效古二首:
入門紫鴛鴦，金井雙梧桐。清歌弦古曲，美酒沽新豐。
效古二首:
快意且為樂，列筵坐群公。光景不可留，生世如轉蓬。
效古二首:
早達勝晚遇，羞比垂釣翁。
效古二首:
自古有秀色，西施與東鄰。蛾眉不可妒，況乃效其顰。
效古二首:
所以尹婕妤，羞見邢夫人。低頭不出氣，塞默少精神。
效古二首:
寄語無鹽子，如君何足珍。

3. 擬古十二首
   作者: 李白著
   內容:
擬古十二... :
青天何歷歷，明星如白石。黃姑與織女，相去不盈尺。
擬古十二... :
銀河無鵲橋，非時將安適。閨人理紈素，遊子悲行役。
擬古十二... :
瓶冰知冬寒，霜露欺遠客。客似秋葉飛，飄颻不言歸。
擬古十二... :
別後羅帶長，愁寬去時衣。乘月托宵夢，因之寄金徽。
擬古十二... :
高樓入青天，下有白玉堂。明月看欲墮，當窗懸清光。
擬古十二... :
遙夜一美人，羅衣沾秋霜。含情弄柔瑟，彈作陌上桑。
擬古十二... :
弦聲何激烈，風捲繞飛梁。行人皆躑躅，棲鳥起回翔。
擬古十二... :
但寫妾意苦，莫辭此曲傷。願逢同心者，飛作紫鴛鴦。
擬古十二... :
長繩難系日，自古共悲辛。黃金高北斗，不惜買陽春。
擬古十二... :
石火無留光，還如世中人。即事已如夢，後來我誰身。
擬古十二... :
提壺莫辭貧，取酒會四鄰。仙人殊恍惚，未若醉中真。
擬古十二... :
清都綠玉樹，灼爍瑤台春。攀花弄秀色，遠贈天仙人。
擬古十二... :
香風送紫蕊，直到扶桑津。取掇世上豔，所貴心之珍。
擬古十二... :
相思傳一笑，聊欲示情親。
擬古十二... :
今日風日好，明日恐不如。春風笑于人，何乃愁自居。
擬古十二... :
吹簫舞彩鳳，酌醴鱠神魚。千金買一醉，取樂不求餘。
擬古十二... :
達士遺天地，東門有二疏。愚夫同瓦石，有才知卷舒。
擬古十二... :
無事坐悲苦，塊然涸轍魚。
擬古十二... :
運速天地閉，胡風結飛霜。百草死冬月，六龍頹西荒。
擬古十二... :
太白出東方，彗星揚精光。鴛鴦非越鳥，何為眷南翔。
擬古十二... :
惟昔鷹將犬，今為侯與王。得水成蛟龍，爭池奪鳳凰。
擬古十二... :
北斗不酌酒，南箕空簸揚。
擬古十二... :
世路今太行，回車竟何托。萬族皆凋枯，遂無少可樂。
擬古十二... :
曠野多白骨，幽魂共銷鑠。榮貴當及時，春華宜照灼。
擬古十二... :
人非昆山玉，安得長璀錯。身沒期不朽，榮名在麟閣。
擬古十二... :
月色不可掃，客愁不可道。玉露生秋衣，流螢飛百草。
擬古十二... :
日月終銷毀，天地同枯槁。蟪蛄啼青松，安見此樹老。
擬古十二... :
金丹寧誤俗，昧者難精討。爾非千歲翁，多恨去世早。
擬古十二... :
飲酒入玉壺，藏身以為寶。
擬古十二... :
生者為過客，死者為歸人。天地一逆旅，同悲萬古塵。
擬古十二... :
月兔空搗藥，扶桑已成薪。白骨寂無言，青松豈知春。
擬古十二... :
前後更歎息，浮榮安足珍。
擬古十二... :
仙人騎彩鳳，昨下閬風岑。海水三清淺，桃源一見尋。
擬古十二... :
遺我綠玉杯，兼之紫瓊琴。杯以傾美酒，琴以閑素心。
擬古十二... :
二物非世有，何論珠與金。琴彈松裏風，杯勸天上月。
擬古十二... :
風月長相知，世人何倏忽。
擬古十二... :
涉江弄秋水，愛此荷花鮮。攀荷弄其珠，蕩漾不成圓。
擬古十二... :
佳人彩雲裏，欲贈隔遠天。相思無由見，悵望涼風前。
擬古十二... :
去去複去去，辭君還憶君。漢水既殊流，楚山亦此分。
擬古十二... :
人生難稱意，豈得長為群。越燕喜海日，燕鴻思朔雲。
擬古十二... :
別久容華晚，琅玕不能飯。日落知天昏，夢長覺道遠。
擬古十二... :
望夫登高山，化石竟不返。

4. 感興六首
   作者: 李白著
   內容:
感興六首:
瑤姬天帝女，精彩化朝雲。宛轉入宵夢，無心向楚君。
感興六首:
錦衾抱秋月，綺席空蘭芬。茫昧竟誰測，虛傳宋玉文。
感興六首:
洛浦有宓妃，飄颻雪爭飛。輕雲拂素月，了可見清輝。
感興六首:
解佩欲西去，含情詎相違。香塵動羅襪，綠水不沾衣。
感興六首:
陳王徒作賦，神女豈同歸。好色傷大雅，多為世所譏。
感興六首:
裂素持作書，將寄萬里懷。眷眷待遠信，竟歲無人來。
感興六首:
征鴻務隨陽，又不為我棲。委之在深篋，蠹魚壞其題。
感興六首:
何如投水中，流落他人開。不惜他人開，但恐生是非。
感興六首:
十五游神仙，仙游未曾歇。吹笙坐松風，泛瑟窺海月。
感興六首:
西山玉童子，使我煉金骨。欲逐黃鶴飛，相呼向蓬闕。
感興六首:
西國有美女，結樓青雲端。蛾眉豔曉月，一笑傾城歡。
感興六首:
高節不可奪，炯心如凝丹。常恐彩色晚，不為人所觀。
感興六首:
安得配君子，共乘雙飛鸞。
感興六首:
嘉谷隱豐草，草深苗且稀。農夫既不異，孤穗將安歸。
感興六首:
常恐委疇隴，忽與秋蓬飛。烏得薦宗廟，為君生光輝。

感遇四首:

感遇四首:

感遇四首:

感遇四首:

感遇四首:

感遇四首:

13. 上崔相百憂章
   作者: 李白著
   內容:
上崔相百... :
共公赫怒，天維中摧。鯤鯨噴蕩，揚濤起雷。魚龍陷人，成此禍胎。火焚昆山，玉石相硾。仰希霖雨，灑寶炎煨。
上崔相百... :
箭發石開，戈揮日回。鄒衍慟哭，燕霜颯來，微誠不感。
上崔相百... :
猶縶夏台。蒼鷹搏攫，丹棘崔嵬。豪聖凋枯，王風傷哀。
上崔相百... :
斯文未喪，東嶽豈頹。穆逃楚難，鄒脫吳災。見機苦遲，二公所咍。驥不驟進，麟何來哉。星離一門，草擲二孩。
上崔相百... :
萬憤結習，憂從中催。金瑟玉壺，盡為愁媒。舉酒太息，泣血盈杯。台星再朗，天網重恢。屈法申恩，棄瑕取材。
上崔相百... :
冶長非罪，尼父無猜。覆盆倘舉，應照寒灰。

14. 萬憤詞投魏郎中
   作者: 李白著
   內容:
萬憤詞投... :
海水渤潏，人罹鯨鯢。蓊胡沙而四塞，始滔天于燕齊。
萬憤詞投... :
何六龍之浩蕩，遷白日于秦西。九土星分，嗷嗷棲棲。
萬憤詞投... :
南冠君子，呼天而啼。戀高堂而掩泣，淚血地而成泥。
萬憤詞投... :
獄戶春而不草，獨幽怨而沉迷。兄九江兮弟三峽，悲羽化之難齊。穆陵關北愁愛子，豫章天南隔老妻。
萬憤詞投... :
一門骨肉散百草，遇難不復相提攜。樹榛拔桂，囚鸞寵雞。舜昔授禹，伯成耕犁。德自此衰，吾將安棲。
萬憤詞投... :
好我者恤我，不好我者何忍臨危而相擠。子胥鴟夷，彭越醢醯。自古豪烈，胡為此繄。蒼蒼之天，高乎視低。
萬憤詞投... :
如其聽卑，脫我牢狴。儻辨美玉，君收白珪。

觀元丹丘... :

39. 南奔書懷
   作者: 李白著
   內容:

南奔書懷:
欃槍掃河洛，直割鴻溝半。歷數方未遷，雲雷屢多難。
南奔書懷:
天人秉旄鉞，虎竹光藩翰。侍筆黃金台，傳觴青玉案。

南奔書懷:
